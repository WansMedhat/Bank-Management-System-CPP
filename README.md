# Bank Management System

A complete Bank Management System developed in C++ using Object-Oriented Programming (OOP) principles.

## Features

### Client Management

* Add New Client
* Update Client Information
* Delete Client
* Find Client
* View All Clients

### Transactions

* Deposit
* Withdraw
* Transfer Between Accounts
* Total Balances Report
* Transfer Log

### User Management

* Add New User
* Update User Permissions
* Delete User
* Find User
* View Users List

### Authentication & Security

* Login System
* Login Register History
* User Permissions

### Currency Exchange

* View Currencies
* Find Currency
* Update Currency Rates
* Currency Calculator

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* File Handling
* Visual Studio
* STL

## Project Structure

The project is divided into multiple modules for better maintainability and scalability.
